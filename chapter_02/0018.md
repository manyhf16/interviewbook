# 数据库表(Test)结构如下
| id | name | age | MANAGER(所属主管人ID) |
| --- | --- | --- | --- |
| 106 | A | 30 | 104 |
| 109 | B | 19 | 104 |
| 104 | C | 20 | 111 |
| 107 | D | 35 | 109 |
| 112 | E | 25 | 120 |
| 119 | F | 45 | null |
要求:列出所有年龄比所属主管年龄大的人的ID和名字?

# 参考解答
```sql
select t.id,t.name from test t where 
    t.age > (select m.age from test m where m.id=t.manager);
```
