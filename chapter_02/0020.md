# 一般如何处理数据库的ID

# 参考解答

* Oracle 数据库可以采用序列来保证主键的唯一性：

创建序列
```sql
create sequence my_seq;
```
在插入数据时使用序列
```sql
insert into my_table(id,name) values(my_seq.nextval,'姓名');
```

* MySQL 数据库可以采用自增列来实现主键唯一：
创建表时：

```sql
create table mytable(
  id int primary key auto 
);
```
