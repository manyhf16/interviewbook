#　在<插入代码>处写如下代码，正确的是( )
```java
class Test {
	int w, x, y, z;
	public Test(int a, int b) {
		x = a;
		y = b;
	}
	public Test(int a, int b, int c, int d) {
		<插入代码>
		w = d;
		z = c;
	}
}

```

```
A.Test(a,b);
B.x=a,y=b;
C.this(a,b);
D.x=a;y=b;
```

# 参考解答
C,D

# 分析
A：Test(a,b)不能直接使用，故A错；B：语法错误；this可以调用本类中其他的构造方法，this(a,b)调用public Test(int a, int b)为对象属性赋值，故C对；D正确。
