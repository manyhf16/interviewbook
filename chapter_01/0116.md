# 以下选项中循环结构合法的是\( \)

A.
```java
while (int i<7){
　　  i++;
　　  System.out.println("i is "+i);
　　}
```

B.int j=3;
　while(j){
　　 System.out.println(" j is "+j);
　}
C.int j=0;
　for(int k=0; j + k !=10; j++,k++){
　　 System.out.println("j is "+ j + " k is "+ k);
　}
D.int j=0;
　do{
　　 System.out.println( "j is "+j++);
　　 if (j == 3) {continue loop;}
　}while (j<10);
```

# 参考解答

C;

A：while循环条件中i未初始化初值；

B：判断条件中的是boolean值，不能为常数

D：loop标签未指明位置
