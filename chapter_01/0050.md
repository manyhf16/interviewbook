#　关于异常说法正确的是（ ）

```
A. 一旦出现异常，程序就终止了
B. 如果一个方法声明将抛出某个异常，它就必须真的抛出那个异常
C. 在catch子句中匹配异常是一种精确匹配
D. 可能抛出运行时异常的方法是不需要声明异常的
```

# 参考解答

异常出现时，可被catch块捕获，程序会继续执行，故A错；

如果一个方法声明将抛出某个异常，它的意思是方法执行时可能会抛出异常，而不是必须抛出，故B错；

catch语句中匹配是一种类型匹配，而不是精确匹配，C错；

D，正确
