# 我们在 web 应用开发过程中经常遇到输出某种编码的字符，如ISO-8859-1 等，如何输出一个某种编码的字符串？

# 参考解答

```
public String translate(String str){
    String tempStr = "你好";
    try{
        tempStr = new String(str.getBytes("ISO-8859-1"), "GBK");
        tempStr = tempStr.trim();
    }catch (Exception e){
        System.err.println(e.getMessage());
    }
    return tempStr;
}
```
