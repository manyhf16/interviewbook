# 在Java中，如何跳出当前的多重嵌套循环？

# 参考解答

在最外层循环前加label标识，然后用break:label方法即可跳出多重循环，例如：



```java
out: for (int i = 0; i < 5; i++) {
  for (int j = 0; j < 5; j++) {
    System.out.println("i=" + i + " j=" + j);
    if (i == 1 && j == 3) {
      break out;
    }
  }
}
```



输出:


```
i=0 j=0
i=0 j=1
i=0 j=2
i=0 j=3
i=0 j=4
i=1 j=0
i=1 j=1
i=1 j=2
i=1 j=3
```
