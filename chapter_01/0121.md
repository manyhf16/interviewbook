#　下列表达式正确的()

A.byte b = 128;
B.boolean flag = null;
C.float f = 0.9239;
D.long a = 2147483648L;

# 参考解答

D

# 分析
A：byte类型数据范围是-128~127，故A错；
B：boolean 的字面值只有true 和 false，不能为null；
C：0.9239字面量是double类型，不能直接赋值给float变量，float字面量需要加F（也可以是f）
