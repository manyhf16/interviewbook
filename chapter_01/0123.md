# 下面那句话等于第二行的表达式（）

```
public interface Foo{
    int k = 4;
}
```
A. final int k = 4  
B. public int k = 4  
C. static int k = 4   
D. abstract int k = 4

# 参考答案

A,B,C

接口变量声明时，可以省略public, final, static 它们显式写出来完全可以。
