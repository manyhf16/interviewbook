# 字符串操作：如何实现字符串的反转及替换？

# 参考解答

例：
```java
StringBuilder sb = new StringBuilder("12345");
System.out.println(sb.reverse().toString()); // 54321

String str = "aaabbbccc";
System.out.println(str.replaceAll("c{3}", "ddd")); // aaabbbddd
```
