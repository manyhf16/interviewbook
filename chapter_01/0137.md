#　以下代码的输出是( )
```
int i = 1, j = 10;
do {
	if (i++ > --j)
		continue;
} while (i < 5);
```
A. i=6 j=5
B. i=5 j=5
C. i=6 j=4
D. i=5 j=6

# 参考解答

D

# 分析
|循环次数|i|j|i++>--j|
|-|-|-|-|
|第1次|2|9|1 > 9 为false|
|第2次|3|8|2 > 8 为false|
|第3次|4|7|3 > 7 为false|
|第4次|5|6|4 > 6 为false|

最后第4次时退出循环
