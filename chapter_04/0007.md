# JSP的常用指令？

# 参考答案

**1. page指令**

主要用来指定页面的编码，是否忽略EL等，例：

```
<%@ page language="java" pageEncoding="UTF-8"%>
```

**2. include指令**
用来实现页面的静态包含，例：
```
<%@ include file="/WEB-INF/common/headeradmin.jsp" %>
```

**3. taglib指令**
用来导入标签库，例：
```
<%@ taglib prefix="c" uri="http://java.sun.com/jsp/jstl/core"%>
```
