# 解释一下ajax 


# 参考解答

ajax可以用来向服务器发送异步请求，是一种用来实现页面局部刷新的技术。

它的底层是JavaScript对象XmlHttpRequest，它的基本使用步骤如下：

```js
// 1. 创建xhr对象
var xhr = new XMLHttpRequest();
// 2. 定义响应返回事件 (回调函数)
xhr.onload = function () {
  // 5. 处理响应文本
  xhr.responseText
}			
// 3. 准备请求
xhr.open('get', '/ssh/comment?content=' + value, true);
// 4. 发送请求
xhr.send();  		
}

```
