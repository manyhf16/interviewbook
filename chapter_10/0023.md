#　Java 编程,打印昨天的当前时刻。

# 参考解答

使用Calendar类：
```java
Calendar cal = Calendar.getInstance();
cal.add(Calendar.DATE, -1);
System.out.println(cal.getTime());
```

使用jdk8：
```java
LocalDateTime today = LocalDateTime.now();
LocalDateTime yestoday = today.plusDays(-1);
System.out.println(yestoday);
```
